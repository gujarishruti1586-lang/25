# FlowApp Landing Page

A polished animated SaaS landing page built with HTML and Tailwind CSS CDN.

## Features
- Glassmorphism navigation with responsive mobile menu
- Advanced animated hero section
- Bento-style feature grid
- Testimonial cards with hover motion
- Gradient CTA section and dark footer

## Run Locally
Open index.html in a browser, or serve the folder with a simple static server.

Example:
```bash
python -m http.server 8000
```
Then visit http://localhost:8000/
